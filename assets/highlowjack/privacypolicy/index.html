YOUR INFORMATION

High Low Jack tracks three key pieces of identifying information: the user provided display name (“username”), a server assigned client id ("client id"), and a client generated unique identifier (“client guid”). All pieces of information are only used in the process of providing the High Low Jack service and experience, and none are sold or shared with third parties.

The client id and client guid are used by the client when communicating with the service to uniquely identify the client across play sessions for the purpose of enabling a user to (re-)connect to a session. The client id and client guid are not communicated to other users and are only known by the originating client and the server. The user can reset their client id and client guid if they wish. If the user plays on mobile, reinstalling the game client will generate a new client guid. If the user plays on the website, deleting the browser data for the site will generate a new client guid. By reseting the client guid, the server will assign a new unique client id the next time the user connects to the server.

The username that the user types in prior to joining or creating a game is known by the originating client, the server, and the other clients that join the same High Low Jack match. The username is associated with the client on a per-match basis. The other clients only retain the username for the duration of the shared match and do not store it. Changing the username between matches will not change the previously provided username as applied to a previous match.

The server stores both pieces of information in log files. These log files are used for diagnostic purposes and for generating statistics about the use of High Low Jack. For diagnostic investigation the client id, client guid, and username are used for identifying and investigating issues. For statistics, the client guid is used to approximate the number of unique clients using the game. Log files are currently stored indefinitely.
